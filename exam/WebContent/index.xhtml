Welcome Admin

How to upload question?

To upload a question goes to the left panel where you find the link upload question. Click on that, it will show you several fields. In the first one put your question, and after that put four options and selects the right option. Now click on the create button. It shows a successful message on the top of the page. Now your question has been created. You can find your question under question list of create question paper section.

How to create question paper?

Go to the left panel and click on the create question paper link, there you could find several fields. Put class (eg: BSC, Hons., Science) at the first field from the drop down list. After that select a batch (eg: 2015, 2016) for whom you creating the paper. After that mention your exam name (eg: test 1). Then select the date of exam from the calendar. After that choose your time of exam, then mention paper number (eg: 001, 002, 003), remember to make it unique. The paper code must be unique all the time. Try to serialize this number. This could be alpha numeric. Don’t use any special character (like: !@#$%^*). After these steps select question by checked the box on left hand side. Select at least 5 question to make a question paper.

Where to find the security key?

A security key is auto generated with the question paper. You can find the question paper at the question paper list at the left panel. This secret key is needed to give exam. Give this to your students for access exam. If a secret key is not working then contact to service provider. It looks like: 3f33tey322la6353hg this.

How to login as member?

Enter valid username and password in the given field. The registered user can make login to perform examination. Please ensure the user is registered with the valid information. If any student not registered, will not able to login.

How to get in touch with support team?

At any point of stuck feel free to contact support team. Go to the contact section on the left panel of window. There you get how to reach support team. You can mail the issue with proper subject mentioned with the mail. Send it to support@comettech.co or any other mail id mention over there.

If question not showing?

If a question not uploaded successfully will not visible to the question list. Make sure to fill all the mandatory fields to upload a question. In case of not uploading question problem contact to support team.

When page not found arises?

Check your internet connection first. This issue entitled due to no network. If this occurs several times then clear cache from the browser settings and restart your browser. If not fixed then try other browser.

What is “exam already given”?

A student can give a particular exam for one time only. If a student try to give exam more than one time then this issue arises. Try to ensure you haven’t given this exam earlier.

If not listed question paper?

If a question paper created by filling all mandatory field then this problem will fixed. Try to fill all the require information to create question paper. If not listed paper yet then contact support team for assistance.

Problem during login as administrator?

Administrator id and password is fixed from the development team. An admin can’t create his/her own user id from the registration page. If need to change the user id of admin then contact to support team. Problem on admin login may be due to network. Try to reconnect your network cable and restart the browser.

What is exam date?

The exam date is set at the time of question paper set. A paper is available for certain period of time. After the mention date the question paper will be deleted automatically from the website. To extend the validity of a question paper contact to support team.

When exam time not working properly?

Exam time is set at the time of paper set. Please ensure proper time to particular paper. After the time the exam will automatically stopped. If timer not working then try to restart your machine, and before that check the system time and date. If yet not fixed then contact to support provider.

Registration failed?

Commonly network failure occur this issue. Check your network cable and try to register again. If no fixed yet then contact to support team.

Wrong email or password?

At the time of login username and password is validated and authenticated from server. Please ensure you typed correct email and password which used at the time of registration. If a valid user or a running user unable to connect then try to clear the cache from your browser settings and then retry. If yet not solved contact to service provider / support team.

Minimum question validation means?

A question paper require at least 5 question to create. Please be sure to select minimum 5 question to make a question paper. This will give you permission to make a paper till you select at least five.
